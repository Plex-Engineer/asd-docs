The ASD SDK provides a set of tools to help you build applications that use the ASD protocol. The SDK assists with interacting with the ASD protocol smart contracts in a Typescript environment.
With the SDK, you can query data from the EVM, encode and decode data, and sign transactions related to the ASD protocol.

## Installation

```bash
npm i {asd-package name}
```
